## 배달
### 다익스트라 알고리즘
- 그래프의 한 정점에서 다른 정점들까지의 **최단 거리**를 구하는 알고리즘. ex) 내비게이션
- 가중치가 있는 그래프에서 동작
#### 주요 아이디어
1. 시작점에서 가장 가까운 정점 선택
2. 선택한 정점에서 인접한 정점으로 가는 최소 비용 업데이트
3. 모든 정점에 대해 반복 -> 최단 거리 확정
### 풀이법
1. `graph`: 각 마을과 연결된 도로 정보 저장
2. 최단 거리 계산: 다익스트라 알고리즘 활용
